# gastos_personales_dataset_generator
Este proyecto busca cubrir la necesidad de un dataset natural de gastos personales de una persona que gana sueldo mínimo y que no tiene dependientes y que tiene opción de hacer horas extra en su empleo para solventar su economía.
 
 Debido a que son datos sencibles es imposible encontrar una fuente pública con la que podamos cubrir la necesida de ingesta de algun proyecto de esta índole; la estrategia que utilizaremos será proyectar datos exitententes de encuestas realizadas en diversos paises como Perú , México y España con el fin de proveer diversos ambitos.
